# url-shortener

CRUD сервис, с помощью которого можно сокращать ссылки, заменяя их на алиас.

## Описание

Пользователь может создавать кастомный алиас на ссылку и записывать это в БД.

## Возможности

- Добавить ссылку и алиас к ней
- Удалить ссылку с алиасом
- Посмотреть ссылку по алиасу
- Обновить ссылку по алиасу

## Установка

```bash
git clone https://github.com/MegaTemik/url-shortener
cd url-shortener
```

## Запуск

```bash
cp .env.example .env
docker compose up
```
